
# Weather API

Simple Weather api usecase for get Weather of current location and with search of city.


## Demo

https://weather-react-vite.netlify.app

## Installation

Install my-project with npm

```bash
  npm install 
  cd weather
  npm run dev
```

## Features

- tailwind css
- Responsive design
- typescript
- react + vite
- openweather api
